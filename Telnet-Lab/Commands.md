## Topology Overview

This topology consists of:
- A **Router (R1)** with IP address: 1.1.1.222 on GigabitEthernet 0/0.
- A **Switch (SW-A)** with IP address: 1.1.1.1 on VLAN 1.
- PCs and a server for testing Telnet access.
